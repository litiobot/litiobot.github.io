Bot de moderación/diversión ágil y fácil de configurar con simples sistemas de anti links y moderación, además de una pizca de IA. ¡Mantén tu servidor seguro y controlado :)!

Características

Moderación

Modera usuarios usando /moderar, donde aparecerá un simple panel con opciones de: baneo, kick, aislamiento entre otros; También tienes un sistema fácil y controlable de anti-links.

Base de datos

Usamos MongoDB para obtener y trasladar pocos datos.

Fácil y sencillo de configurar
